## What this chart installs

`loki-stack` is an umbrella chart that can deploy:

- **Loki** (logs storage/query) via the `grafana/loki` subchart
- **Grafana Alloy** (log collection) via the `grafana/alloy` subchart
- **Prometheus + Grafana** via the `prometheus-community/kube-prometheus-stack` subchart
- **RustFS** (optional S3-compatible object storage) via the `rustfs/rustfs` subchart when `rustfs.enabled=true`

Artifact Hub install page for this umbrella chart:

- **RustFS** (`rustfs/rustfs`): `https://github.com/rustfs/helm` (optional, disabled by default)

## Install chart

### From Helm repository

Add the Helm repository:

```bash
helm repo add helm-charts https://rishang.github.io/helm-charts/charts
helm repo update
```

Install or upgrade:

```bash
helm upgrade --install loki-stack helm-charts/loki-stack --namespace monitoring --create-namespace
```

### From local path

```bash
helm upgrade --install loki-stack ./helm/loki-stack --namespace monitoring --create-namespace
```

### Getting Started

```yaml
# values.yaml
loki:
  singleBinary:
    persistence:
      enabled: true
      size: 10Gi
      storageClass: default

kube-prometheus-stack:
  grafana:
    adminUser: admin
    adminPassword: admin
    persistence:
      enabled: true              # no PVC, data resets on pod restart
      storageClass: default
      size: 1Gi

```

Install or upgrade the chart:

```bash
helm upgrade --install loki-stack helm-charts/loki-stack --namespace monitoring --create-namespace --values values.yaml
```

## Upgrade Chart

From repository:

```bash
helm upgrade loki-stack helm-charts/loki-stack --namespace monitoring --values values.yaml
```

## Delete Chart

```bash
helm uninstall loki-stack --namespace monitoring
```
